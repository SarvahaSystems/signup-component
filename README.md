# **Project Setup**

## **Client (React Application)**

### **Install Dependencies**

```bash
cd client/
npm install or
yarn install
```

### Run the Client

```bash
npm start or
yarn start
```

The application will be available at http://localhost:3000/ by default.

## **Server (Node.js Application)**

### **Install Dependencies**

```bash
cd server
npm install or
yarn install
```

### Run the Server

```bash
npm start or
yarn start
```

The server will be running at http://localhost:3001/ by default.
